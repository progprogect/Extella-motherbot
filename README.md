# Extella Motherbot

Multi-tenant Telegram bot constructor powered by Extella AI.

## Features
- Create your Telegram bot in 2 minutes
- AI Assistant (GPT-4)
- Translator
- Content Generator
- Runs 100% on Railway — no server needed

## Deploy
1. Fork this repo
2. Connect to Railway
3. Add PostgreSQL plugin
4. Set env vars (see .env.example)
5. Done!
